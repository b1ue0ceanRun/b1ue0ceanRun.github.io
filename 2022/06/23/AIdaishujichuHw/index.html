关联矩阵(The Incidence Matrix)

图1展示了一个边数m=6和节点n=4的图。从6乘4的矩阵A中我们可以得知哪些节点是由哪些边连接的。第一行-1, 1, 0, 0表示第一条边从node①指向node②（-1表示节点1，因为箭头向外，+1表示节点2，箭头向内）。
A中的行号是边的编号，列号1、2、3、4是节点编号!

可以通过看图来写出他的关联矩阵。第二张图有四个node但只有三条边，他的关联矩阵为3x4

​ 图2

这是两个极端的情况。边缘的最大数量是½n(n - 1) = 6，保持连接的最小数量是n - l = 3。 消除法将每个图都减少为一棵树。循环在A中产生依赖行，在梯形图U和R中产生零行。看看第一个图中边1、2、3的大循环，它导致U中的零行。

第一张图每一个节点都有边相连接，第二张图是一个树状图没有闭环。这是两种极端的情况。边缘的最大数量是½n(n - 1) = 6 , 保持连接的最小数量是n - l = 3。

用消去法可以将任何一张图变成树状图。环状结构导致A中行相关，可以转化为行阶梯矩阵。看看第一个图中边1、2、3的大闭环，它导致出现零行。

这些是很典型的步骤，当edge① 和edge②都通过node① 时，消去法产生了没有通过节点①的 “shortcut edge”（此时指edge3）。如果图已经存在”shortcut edge” 并形成了一个闭环，使用消去法将得到零行。最终我们得到一个树状图。

一个想法自然而然地出现了：当边形成一个循环时，行是依赖性的。

一个小例子：当x1, x2, x3, x4为节点的电压时，Ax给出电压差:

  1. 零空间包含了所有 Ax = 0 的解。如果所有的六个电压差都是零，这意味着所有的四个电压都是相等的。零空间中的每一个x都是一个常数向量：x=（c，c，c，c）。A的零空间是R n-its 维空间中的一条线，其维度为n - r = l。第二个关联矩阵B也有同样的零空间，它包含（1，1，1，1）。

我们可以对所有电压提高或降低相同的电压c，而使得结果并没有发生改变。电压中存在一个 “任意常数”。我们可以通过C来提高或降低一个函数，而不改变其导数。

微积分在不确定的积分上加上 “+C”。向量x加上了( c, c, c, c) ，线性代数在Ax=b的一个特定解中加入了零空间中的任何向量Xn。

同样地，当我们固定x4=0时，零空间也消失了。未知的x4被移除，A和B的第四列也被移除（这些列乘以x4）。电气工程师会说，节点4已经被 “接地 “了。

2.行空间包含六行的所有组合形式，它的维数当然不是六。 等式 r + (n - r) = n 为 3+1=4。矩阵的秩 r = 3, 同我们从消去法中得到的相同。三条边之后，我们将形成闭环，新的(多出的)行是非相关的。 我们如何判断 v = (v1, v2, v3, v4) 是不是在行空间内？比较慢的方法是合并矩阵的行，比较快的方法是通过正交：

v is in the row space if and only if it is perpendicular to ( 1, 1, 1, 1) in the nullspace

v 在行空间内当且仅当它垂直于零空间中的( 1, 1, 1, 1)

The vector v = (0, 1, 2, 3) fails this test-its components add to 6. The vector (-6, 1, 2, 3) is in the row space: -6+ 1 + 2+3 = 0. That vector equals 6(row 1) +5(row 3) +3(row 6). Each row of A adds to zero. This must be true for every vector in the row space.

3.列空间包含四列的所有组合。我们期望有三个非相关的列，因为有三个非相关的行。A的前三列是非相关的（任选三列也是如此）。但是这四列加起来就是零向量，这又说明（1，1，1，1）是在零空间内。我们如何判断一个特定的向量b是否在关联矩阵的列空间中？

回答一：尝试解决Ax=b，这就错过了所有的洞察力。和以前一样，正交性给出了一个更好的答案。我们现在正在研究基尔霍夫的两个著名的电路理论定律理论–电压定律和电流定律（KVL和KCL）。这些都是自然的表达线性代数的 “定律”。看到左零空间的起到关键作用，特别令人高兴。

回答二：Ax是电压差Xi - Xj的矢量。如果我们围绕着图形中的一个闭环添加差值，它们会被抵消，留下零。在边1,3,-2 (the arrow goes backward on edge 2)形成的大三角形差值抵消了。

Sum of differences is 0

Kirchhoff’s Voltage Law: The components of Ax = b add to zero around every loop.

通过测试每个环，电压法决定b是否在列空间中。Ax = b 可以被精确地解出当b的组件满足所有与A的行相同的依赖关系时。那么消除法会得出0=0，Ax=b是一致的。

4.左零空间包含A^T y = 0的解，其维度为m - r = 6 - 3。

真正的方程数是r=3而不是n=4。理由:四个方程相加为 0=0。第四个方程是由前三个方程自动产生的。这些方程是什么意思？第一个方程说，-y1-Y2-y4=0。进入节点1的净流量为零。第四个方程说，y4 + y5 + Y6 = 0。节点4的流量减去流出的流量为零。方程A^T y = 0是著名的和基本的。

Kirchhoff’s Current Law: A^t y = 0 Flow in equals flow out at each node.

This law deserves first place among the equations of applied mathematics. It expresses “conservation” and “continuity” and “balance.” Nothing is lost, nothing is gained. When currents or forces are balanced, the equation to solve is A T y = 0. Notice the beautiful fact that the matrix in this balance equation is the transpose of the incidence matrix A

这一定律在应用数学的方程中应占首位。它表达了 “保存”、”连续 “和 “平衡”。没有失去什么，也没有得到什么。当 当电流或力量平衡时，要解决的方程是A^T y = 0。注意一个美丽的事实–这个平衡方程中的矩阵是关联矩阵A的转置。

A^T y = 0的实际解是什么？电流必须自行平衡。最简单的方法是绕着一个圈流动。如果一个单位的电流绕过大三角形 (1和3边向前走，2边向后走），六个电流为y=（l, -1, 1, 0, 0, 0）。这就满足了ATy=0。每个环形电流都是电流定律的解。流入 等于每个节点的流出量。一个较小的循环在边1上前进，在5上前进，在4上后退。在4上。那么y = (1, 0, 0, -1, 1, 0)也是在左边的空域。我们期待三个独立的y’s: m-r=6-3=3。图中的三个小循环是独立的。大三角形似乎给出了第四个 y，但这个流量是小环的流量之和。围绕小环的流量之和。围绕3个小环的流动是左边零空间的一个基。

对于平面内的每一个图形，线性代数都会产生欧拉公式。拓扑学中的定理1!

(number of nodes) - (number of edges) + (number of small loops) = 1.

这就是（n）-（m）+（m-n+1）=1。我们例子中的图形有4 - 6 + 3 = 1。

一个的三角形有（3个节点）-（3条边）+（I个循环）。在一个有9条边的10个节点的树上(没有闭环)，欧拉的计算为 10 - 9 + 0 。所有的平面图都可以得出答案为1。

下图是一个有电流源的网络。基尔霍夫电流定律的变化 从A^T y = 0到A^T y = f，以平衡来自外部的源f。流入每个节点的电流仍然等于流出的电流。六条边会有电导率c1, …. ，c6，而电流 源进入节点1。源从节点4出来，以保持整体平衡 (in= out)。问题是：找到六条边上的电流y1, –, y6。现在网络中的流动使我们从关联矩阵A到拉普拉斯矩阵A^T A。

Voltages and Currents and A^T Ax = f

我们从节点上的电压x = ( x1, … , Xn ) 开始。到目前为止，我们用Ax来沿着边寻找电压差Xi - Xj。 我们有电流法A^T y = 0来寻找边的电流y = (y1, … Ym)–如果网络中的所有电阻都是1，欧姆定律将与之匹配 y = Ax。那么A^T y = A^T Ax = 0。我们已经很接近了，但还没有达到目的。如果没有任何来源，A^T Ax = 0的解决方案将没有流动：x = 0和y = 0。
我可以看到有三种方法可以产生x!=0和y!=0。

  1. 给一个或多个节点分配固定电压Xi。
  2. 在一个或多个边上添加电池（电压源）。
  3. 添加进入一个或多个节点的电流源 See Figure 10.2

Networks and transport(A)CA

在一个真实的网络中，沿着一条边的电流y是两个数字的乘积。一个数字是边缘两端的电位x之差。这个电压差 是Ax，它驱动电流。另一个数字c是 “电导率”–它衡量流量通过的容易程度–衡量流量通过的容易程度。在物理学和工程学中，c的值是由材料决定的。对于电流来说，金属的c值相对是高的，而塑料则相对较低。对于超导体来说，c几乎是无限大的。如果我们 考虑到弹性拉伸，金属的c可能很低，塑料的c可能更高。

在经济学上，可以用c来衡量一条边的容量或其成本。总而言之，图可由它的关联矩阵A得知。这告诉我们结点与结点之间的联系。

一个网络更进一步，给每个边分配一个电导率c。这些数字c1, … Cm进入 “电导矩阵 “C–它是对角矩阵。对于电阻网络来说，电导率是c = 1 / （电阻）。除了用于整个电流系统的基尔霍夫定律外，我们还有用于每个电流的欧姆定律。欧姆定律将边1上的电流y1与电压差x2 - x1联系起来。

Ohm’s Law: Current along edge = conductance times voltage difference.

所有m电流的欧姆定律是y = -CAx。矢量Ax给出电位差，C乘以电导率。将欧姆定律与基尔霍夫定律相结合.
